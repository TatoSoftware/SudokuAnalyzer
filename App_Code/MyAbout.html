✧Analyze any Sudoku puzzle.

✧Pick a puzzle for me or Create my own scenario.

  • Try the following process to ensure maximum fun, minimum time and plenty of challenges.
  • To start select scenario you want to play by clicking on Start New
  • Click Solve then 16 Iterations 2 or 3 times to solve the easy bits at the start of any new puzzle.
  • Click Show Hints. This will highlight a few cells where some candidates can be removed.
  • If you have made many changes click Check It! to make sure the puzzle is still solveable.
  • Click anywhere in the right panel to remove highlighting from the puzzle.
  • To enter a number for a cell click on the top row below.
  • To Pencil in a possible number click on the bottom row below.

✧This app contains only a sophisticated Sudoku Solving engine.

✧Advanced Techniques implemented in order of difficulty are: X-Wing, Swordfish, Co-joined Chains, Coloring, Gordonian Rectangle, One-Sided Gordonian Rectangle, Gordonian Polygon Plus and Nishio.  A brute force algorithm is used to ensure a solution does exist and to check for multiple solutions.   Peter Gordon's book 'Mensa Guide to Solving Sudoku'
